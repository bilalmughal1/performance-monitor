# AGENTS.md

This file provides instructions for AI agents working on this repository.

## General Principles

*   **Follow the existing code style.**
*   **Write clear and concise code.**
*   **Add comments** where necessary.
*   **Write tests** for your changes.
*   **Update the documentation** if necessary.
*   **Follow the guidelines in `CONTRIBUTING.md`**.

## Project Structure

*   `src/app`: Contains the routes for the application.
    *   `(auth)`: Contains the authentication routes.
    *   `dashboard`: Contains the dashboard routes.
*   `src/components`: Contains the React components.
    *   `common`: Contains the general UI components.
    *   `dashboard`: Contains the dashboard-specific components.
*   `src/lib`: Contains the utility functions and libraries.

## Testing

*   Run the tests using `npm test`.
*   Make sure all tests pass before submitting a pull request.

## Pull Requests

*   Create a pull request from your fork to the main repository.
*   Make sure your pull request has a clear and descriptive title and description.
*   Make sure your pull request passes all the checks.
